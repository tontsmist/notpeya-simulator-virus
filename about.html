📖 About NotPetya

NotPetya is a destructive malware that appeared in June 2017.

Disguised as ransomware, it actually aimed to wipe systems by encrypting the Master File Table (MFT) and overwriting the Master Boot Record (MBR).

It spread using the EternalBlue exploit and other legitimate tools like PsExec.

The main target was Ukraine, but it quickly infected thousands of systems globally.
